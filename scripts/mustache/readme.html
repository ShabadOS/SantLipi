Sant Lipi

Sant Lipi is a variable font with the purpose of fully expressing the unique characters and combos found in the database at Shabad OS.

Please see Sant Lipi's GitHub repo to learn more.

Usage

Desktop - Select all files in the ttf folder, right-click, and either "Install for all users" (Windows) or "Open with Font Book/Viewer" (macOS/Linux). It is recommended to uninstall any previous versions of Sant Lipi before installing a new version.

Web - WOFF2 is the same as TTF but compressed for lower bandwidth usage. It is supported by all modern browsers (see "Can I Use WOFF2?"). See how a variable font is defined in qa/style.css. Learn more about WOFF2 on web.dev.

Quality Assurance

There is a Sant Lipi Test Suite to help confirm the rendering on various platforms/engines. See qa/index.html. You may also submit feedback or report issues using the link at the bottom of that page.
